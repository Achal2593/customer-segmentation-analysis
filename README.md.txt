# Customer Segmentation using RFM Analysis

This project analyzes customer purchasing behavior using the Online Retail dataset.

Techniques used:
- Data Cleaning
- RFM Analysis
- K-Means Clustering

Tools:
Python, Pandas, Scikit-learn, Matplotlib 

## Dataset

The dataset used in this project is the Online Retail dataset from Kaggle.

Download dataset here:
https://www.kaggle.com/datasets/vijayuv/onlineretail

After downloading, place the file in the `data` folder.
