# Customer Segmentation using RFM Analysis

## Project Overview
This project analyzes customer purchasing behavior using the Online Retail dataset and segments customers using RFM analysis and K-Means clustering.

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. RFM Feature Engineering
4. K-Means Clustering
5. Customer Segmentation
